## 🗄️ Database Schema

### Users Table
```sql
CREATE TABLE users (
    id INTEGER PRIMARY KEY,
    username VARCHAR(50) UNIQUE,
    email VARCHAR(100) UNIQUE,
    hashed_password VARCHAR(255),
    is_admin BOOLEAN DEFAULT FALSE,
    created_at TIMESTAMP
);
```

### Mindmaps Table
```sql
CREATE TABLE mindmaps (
    id INTEGER PRIMARY KEY,
    user_id INTEGER FOREIGN KEY,
    title VARCHAR(200),
    content TEXT,
    created_at TIMESTAMP,
    updated_at TIMESTAMP
);
```

## 🔌 API Endpoints

### Auth Endpoints
- POST /api/auth/register
- POST /api/auth/login
- GET /api/auth/me

### Mindmap Endpoints
- GET /api/mindmaps (list user's mindmaps)
- POST /api/mindmaps (create new)
- GET /api/mindmaps/{id} (get one)
- PUT /api/mindmaps/{id} (update)
- DELETE /api/mindmaps/{id} (delete)
- GET /api/mindmaps/{id}/render (render as HTML)

### Admin Endpoints
- GET /api/admin/stats (system statistics)
- GET /api/admin/users (list all users)
- GET /api/admin/mindmaps (list all mindmaps)
